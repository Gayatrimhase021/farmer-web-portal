# 🌾 Farmer Web Portal

## 📌 Project Overview

Farmer Web Portal is a web-based platform developed to help farmers access important agricultural information such as crop details, market rates, and weather updates in a simple and user-friendly way.

The main objective of this project is to reduce the information gap between farmers and digital agricultural services.

---

## 🚜 Features

- 🌱 Crop Information  
- 💰 Market Price Updates  
- 🌦 Weather Updates  
- 📱 Responsive Design  
- 🔐 Login System  

---

## 🛠️ Technologies Used

**Frontend:**  
- HTML  
- CSS  
- JavaScript  

**Backend:**  
- (Optional) Node.js / PHP  

**Database:**  
- MySQL
